# 🚀 DSA Pattern Finder

A smart web app that analyzes DSA problem statements and predicts the most relevant algorithmic pattern with reasoning and solution steps.

---

## 🔥 Live Demo

👉 https://dsa-pattern-finder-gray.vercel.app/

---

## 🧠 What It Does

* 📌 Detects patterns like:

  * Sliding Window
  * Two Pointers
  * Binary Search
  * Dynamic Programming
  * Graph
  * Greedy
  *Prefix Sum

* 📊 Provides:

  * ✅ Predicted Pattern
  * 📈 Confidence Score
  * 🤔 Why this pattern?
  * 🛠️ How to solve (step-by-step thinking)

---

## 🛠️ Tech Stack

**Frontend**

* React (Vite)
* Axios

**Backend**

* Node.js
* Express.js

**Deployment**

* Frontend: Vercel
* Backend: Render

---

## ⚙️ How It Works

1. User enters a DSA problem or URL
2. Backend extracts key signals (keywords, structure)
3. Scoring engine ranks possible patterns
4. Decision engine selects best match
5. Returns explanation + solving approach
